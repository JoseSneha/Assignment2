ERP Dashboard - Retail Industry

Designed and developed an interactive ERP dashboard for the retail industry to monitor procurement and operational performance.

Key Features:

  • Real-time data visualization of procurement metrics.
  • Interactive charts and graphs for purchase orders, goods received, invoices, vendor performance, and lead times.
  • User-friendly interface with clean design for business users.
  • Responsive layout for accessibility on various devices.

Technologies Used:

  • HTML5 & CSS3 for structure and styling.
  • JavaScript for interactivity and data handling.
  • Chart.js for data visualization.
  • Bootstrap for responsive design.
